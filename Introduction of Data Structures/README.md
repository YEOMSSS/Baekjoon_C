## 3주차 실습 배열과 포인터

#### 배열의 초기화

자료형 배열 이름[배열 크기] = { 초기값 리스트 }; 형태로 초기화한다.  
모든 원소에 초기값이 주어지면 배열 크기는 생략 가능

#### 문자형 배열의 초기화

널\0문자에 주의  
조건식에 arr[i]를 넣으면 \0을 만날 때가지 반복한다.

#### scanf로 문자열 입력받기

scanf_s는 sizeof로 문자열을 받아야 한다? scanf는 필요없던데?  
`scanf("%s", str)`
`scanf_s("%s", str, sizeof(str));`
